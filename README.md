### ⬜ 칸반보드 기반의 프로젝트 협업 툴 서비스!

    칸반보드 형식의 프로젝트 협업 툴로, 업무의 시각적 표현을 통해 팀이 작업과 프로세스를 효율적으로 관리할 수 있게 도와줍니다.

## 💬주요기능

- 회원 기능
  - 회원가입, 로그인, 로그아웃 👨‍🦲
- 보드 기능
  - 프로젝트 별 보드 CRUD 🔧
- 컬럼 기능
  - 개발 기능들을 진행 상태 따라 구분 ✅
- 카드 기능
  - 개발 기능에 대한 세부 정보 CRUD 및 댓글을 통한 의견 공유 📃
